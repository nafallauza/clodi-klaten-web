CLODI KLATEN BABYSHOP - VERSI PRODUK BARU FIX

Cara membuka:
1. Extract ZIP.
2. Buka folder clodi-babyshop-tailwind-products-fixed di VS Code.
3. Pastikan internet aktif karena Tailwind dan Bootstrap Icons memakai CDN.
4. Klik kanan index.html.
5. Pilih Open with Live Server.

Perbaikan:
- Logo sudah diganti menjadi file PNG: assets/img/logo.png
- Semua produk sudah diganti memakai gambar baru yang dikirim.
- Nama produk sudah disesuaikan dengan alat/barangnya.
- Product image dibuat object-contain supaya gambar tidak kepotong.
- File tetap dipisah:
  index.html
  css/style.css
  js/main.js
  assets/img/
