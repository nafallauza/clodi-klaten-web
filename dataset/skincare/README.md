# Skincare Dataset

Karena keterbatasan lisensi atau kebutuhan otentikasi Kaggle, dataset skincare tidak dapat diunduh secara otomatis. Silakan unduh dan letakkan dataset tabular di dalam folder ini (`dataset/skincare/`).

Dataset yang diperlukan:
1. **Indonesian Skincare Sample Dataset**
2. **Indonesia Skincare Female Daily Reviews**
3. **19000 Skincare Products Database**
4. **INCI Ingredients Dataset**

Pastikan file berbentuk `.csv` atau `.json`.
Nantinya pipeline kita akan membaca file dari folder ini untuk dilakukan _preprocessing_ dan dimasukkan ke dalam database SQLite.
