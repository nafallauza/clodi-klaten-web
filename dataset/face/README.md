# Face Dataset (Computer Vision)

Dataset ini digunakan untuk melatih dan mengevaluasi model deteksi tingkat keparahan jerawat (Acne) serta jenis kulit wajah.

Dataset yang diperlukan:
1. **ACNE04 Dataset**
2. **Fitzpatrick17k**

Silakan letakkan folder dataset (berisi gambar dan file anotasinya) di dalam folder ini (`dataset/face/`).
Untuk sementara waktu, aplikasi akan berjalan menggunakan _dummy_ inference selama model asli belum dilatih/diunduh.
